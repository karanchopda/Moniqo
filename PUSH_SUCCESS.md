## 🔄 Future Pushes

Now that the repository is set up, future pushes are easier:

### Make Changes:
```bash
# Edit files
# ...

# Stage changes
git add .

# Commit
git commit -m "feat: your feature description"

# Push (will need token again)
git push https://[YOUR_TOKEN]@github.com/karanchopda/Moniqo.git main
```

### Or Set Up Credential Helper:
```bash
# Store credentials permanently
git config --global credential.helper store

# First push will ask for credentials
git push

# Username: karanchopda
# Password: [YOUR_GITHUB_TOKEN]

# Future pushes won't ask
git push
```

## 📱 Clone on Another Machine

To work on another computer:

```bash
# Clone
git clone https://github.com/karanchopda/Moniqo.git
cd Moniqo

# Setup backend
cd backend
npm install
cp .env.example .env
# Edit .env with your credentials
npx prisma generate

# Setup frontend
cd ../frontend
npm install
cp .env.example .env
# Edit .env

# Run
cd ../backend && npm run dev  # Terminal 1
cd ../frontend && npm run dev # Terminal 2
```
